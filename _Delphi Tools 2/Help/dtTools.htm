function DeAlpha(s: string; m: Integer): string;

DeAlpha decodiert wie DeCode den String s Es werden allerdings nur Buchstaben (ohne Umlaute) kodiert.

function DeCode(s: string; m: Integer): string;

DeCode decodiert nach dem ASCII-Zeichensatz einen String und gibt den codierten String zurück. Dabei wird der String s einfach um -m Zeichen verschoben. Diese Funktion ist nicht für den professionellen Einsatz geeignet und kann leicht durch Brute-Force-Attacken gecrackt werden. Siehe auch EnCode

function EnAlpha(s: string; m: Integer): string;

EnAlpha codiert wie EnCode den String s Es werden allerdings nur Buchstaben (ohne Umlaute) kodiert.

function EnCode(s: string; m: Integer): string;

EnCode codiert nach dem ASCII-Zeichensatz einen String und gibt den codierten String zurück. Dabei wird der String s einfach um m Zeichen verschoben. Diese Funktion ist nicht für den professionellen Einsatz geeignet und kann leicht durch Brute-Force-Attacken gecrackt werden. Siehe auch DeCode

function IsDelphi4Running: Boolean;

IsDelphi4Running gibt True zurück , wenn Delphi 4 aktiv ist.

function IsDelphiRunning: Boolean;

IsDelphiRunning gibt True zurück , wenn Delphi aktiv ist, unabhänging von der Version
