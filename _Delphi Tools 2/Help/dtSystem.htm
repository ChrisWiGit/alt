function CompareInt(i1, i2: string): Integer; overload;

CompareInt vergleicht wie CompareStr zwei Integervariablen als Text. Der Rückgabewert ist dergleiche, wie CompareText.

function CompareInt(i1, i2: Integer): Integer; overload;

CompareInt vergleicht wie CompareStr zwei Integervariablen. Der Rückgabewert ist dergleiche, wie CompareText.

procedure GetProperties(Obj: TObject; Items: TStrings);

GetProperties listet alle Eigenschaftsnamen eines Objekts Obj als Strings in Items auf. Items muss vorher erzeugt werden.

function IsValidObject(Obj : TObject) : Boolean;

IsValidObject testet ob eine Objektvariable noch gültig ist (TRUE). Es können auch Objektvariablen übergeben werden , die nicht mehr eindeutig als gültig zu indentifizieren sind mehr zu dem Thema ungülitge Objektvariablen , in der mitgelieferten Textdatei "Objekteigenschaft.txt"

procedure NOP;

NOP ist eine Platzhalterprozedur, um den Delphi-Editor davon abzuhalten, Funktionen aus dem Quelltext zu nehmen. Sie hat sonst keinerlei Funktion.

function SetPoint(x, y: Integer): TPoint;

SetPoint erzeugt einen TPoint-Record mit den angegeben Koordinaten.

function StrIdx(needle : String; stack : array of String) : Integer;

StrIdx macht case-Anweisung auch mit Strings möglich aus PC Magazin August 2001 - Seite 208 Bsp. case strIdx(edit1.Text, ['eins','zwei','drei']) of 0 : listbox1.Itemindex := 0; 1 : listbox1.Itemindex := 1; 2 : listbox1.Itemindex := 2; 3 : listbox1.Itemindex := 3; end;
