Beschreibung

TDialogExt erweitert TDialog um die Möglichkeit, das Dialogende zu bestimmen.

constructor Create(AOwner: TComponent); override;

Create erzeugt den Dialog

function Execute: Boolean; virtual;

Execute stellt den Dialog dar, und liefert TRUE bei OK, und FALSE bei Abbruch zurück.

procedure GetData(var DoClose: Boolean); virtual;

GetData wird nach Dialogschluss (immer) aufgerufen und muss überschrieben werden, wenn die Daten des Dialogs untersucht werden sollen. Weil diese Methode immer bei der Beendigung aufgerufen wird,sollte mit ModalResult die Sachlage erfragt werden : if ModalResult <> mrOk then Exit;

procedure OnDoCloseQuery(Sender: TObject; var CanClose: Boolean); virtual;

OnDoCloseQuery wird aufgerufen, wenn der Benutzer den Dialog beenden will. Überschreiben Sie OnDoCloseQuery, um mit Canclose das Dialogende zu bestimmen.

procedure SetData; virtual;

SetData wird vor der Darstellung des Dialoges aufgerufen, um Kontrollelemente mit Inhalt aufzufüllen. Sie muss dazu überschrieben werden.

function ShowModal: Integer; override;

ShowModal stellt den Dialog modal dar.
