function ReadRGBQuadsToStream(Stream: TStream): TRGBQuads;

ReadRGBQuadsToStream lädt ein RGBQuads Typ aus einen Stream. Der Rückgabewert ist die Anzahl der geladenen Bytes. Hinweis : Zur Nutzung dieser Funktion muss die Compilerdirektive : USEDELPHIX definiert worden sein. $DEFINE USEDELPHIX (in geschweiften Klammern) Siehe auch SaveRGBQuadsToStream

function SaveRGBQuadsToStream(Stream: TStream; RGBQuads: TRGBQuads): Longint;

SaveRGBQuadsToStream speichert ein RGBQuads Typ in einen Stream. Der Rückgabewert ist die Anzahl der gespeicherten Bytes. Hinweis : Zur Nutzung dieser Funktion muss die Compilerdirektive : USEDELPHIX definiert worden sein. $DEFINE USEDELPHIX (in geschweiften Klammern) Siehe auch ReadRGBQuadsToStream.
