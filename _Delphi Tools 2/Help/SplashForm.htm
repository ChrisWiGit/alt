Unit SplashForm

Beschreibung

SplashForm.pas beinhaltet die Klasse TSplashDlg für den Umgang mit dem SplashDialog.

Klassen, Schnittstellen und Objekte

NameBeschreibung
Klasse TSplashDlg TSplashDlg wird benutzt, um eine Informationsanzeige darzustellen, während das Hauptprogramm damit beschäftigt ist Daten zu laden.

Typen

NameBeschreibung
TOnExtraPaint TOnExtraPaint ist das Ereignis, dass aufgerufen wird, wenn etwas auf den Dialog gezeichnet werden soll.

Variablen

NameBeschreibung
SplashDlg SplashDlg wird benutzt, um eine globale Klassenreferenz auf den Splashdialog zu erzeugen. Diese Variable wird normalerweise nicht benutzt.
