function ExecSQL(DSQL: TIBSQL; SQLLines : TStringList; SeperatorChar : Char = ';') : Integer;

ExecSQL führt eine Reihe von SQL Anweisung aus "SQLLines" in einer TIBSQL-Klasse aus. Der Rückgabewert ist -1 im Erfolgsfall, sonst wird eine Exception (EIBInterBaseError) ausgelöst, und die Zeile, in der der Fehler auftrat zurückgeliefert.

function IsBDEInstalled: Boolean;

IsBDEInstalled untersucht ob BDE installiert ist und gibt bei Erfolg TRUE zurück.
